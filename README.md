### 恶毒折磨分阶  
&emsp; AI 研究本来靠脑子的， 结果这些垃圾却靠偷窥阴祟伎俩， 生生将一项科技发挥成偷窥利器， 变成这些垃圾小偷儿谋私利器。 阴渠垃圾阴窃结果。 <br>  
正常的涉己人体的科学研究， 早就有规范的。 比如采集人体电信号， 招募志愿者， 通过当面设置试验条件与输入， 采集相应的人体电信号， 简单的也就几天， 多也不到百天。 但是这些阴窃垃圾， 为了谋私利， 比如骗取钱财， 于是乎将一项试验生生弄成了阴祟伎俩， 绝大多数的资源被巧立名目归结到布置偷窥设施上。 再以偷窥设备不够精确为借口， 不仅大肆折磨目标， 还各种瞎猜， 然后再坑更多钱财。 稍微有点脑子， 一看就能轻易而举发现这些垃圾的花招。 但是， 没有这些小贪咋能供大贪！ 所以层层相护， 才构建了这邪恶化的组织形式。 <br>

模拟信号本就不容易准确， 而这些狗屁的垃圾， 狗屁不懂， 比如偷窥窥探与操控的遥控遥测收发环节根本没形成“闭环反馈控制”。 瞎JB的操作， 纵容这些畜牲肆意地恶毒折磨攻击。 <br>
